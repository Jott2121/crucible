# docs/superpowers/

This directory is the project's dated build log: specs (`specs/`) and plans
(`plans/`) written as each piece of crucible was designed and built, plus
probe results, kept as frozen history alongside `experiments/`.

Instructions and paths inside are dated -- they reflect the state of the
project at the time each doc was written, not current usage. For how to use
crucible today, see the top-level `README.md`.
